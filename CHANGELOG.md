# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-04

### Added

- Initial release of GAC (Git AI Commit).
- Support for OpenAI-compatible APIs and Claude CLI.
- Interactive commit message confirmation.
- Automatic diff summarization for large changes.
- Support for Conventional Commits and simple formats.
- Multi-language support (English, Chinese).
